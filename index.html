DOC-to-TXT

What it is

Doc-to-Txt is a simple app for Mac OS X that batch converts Microsoft Word files (DOC or DOCX) to textfiles. It uses the built-in textutil utility.

How to use it

Download the app from the Github repository (or you can simply download the Applescript). Double-click on the app, choose the file(s) to be converted, choose the destination folder, hit OK and the script will run. When it's done converting the app will close itself.

Limits

There are limits to this app. Plaintext cannot handle special formatting, images, tables, footnotes/endnotes or anything else like that. This is intended as a basic utility to quickly get the text out of your Word files for better permanent storage. I plan on writing a similar script that will convert to RTF, but for now that's not possible.
